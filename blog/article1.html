Lets try out Sblg!

Maybe I can find a Blogging system that will encourage me to write something interesting.
Thomas Willis
2021-05-09

I've been meaning to start Blogging for quite some time, but especially since I decided to drop all Social Media. I needed to have some kind of media that would work with the way I tend to use technolgy. I didn't want to use a blogging service with loads f features and wizbang graphics, nor did I want to have my data managed owned or stored by some unknown entity. Since for the last year I have been attempting to see how much of my daily computer life I could accomplish with OpenBSD on the release of 6.9 I started looking into what was offered in base and the ports.

There wasn't much in ports, but Sblg was exactly what I was looking for. Edit XML templates and compile the whole page at one time including an Atom rss link. The retro template, was just the thing and quickly utilized it for my sites CSS as well. My initial attempts to compile failed horribly, but I copied the repo to my project directory and recompiled to a local directory and then pushed it up to my server with Git and everything just worked. Now to replace the example.xml files with my own ideas and maybe I can stay motivated.
